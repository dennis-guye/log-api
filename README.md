# Tail

## Requirments

- Node - v20.7.0
- Yarn - 1.22.19

## Technologies

- Open Api Specification for clinet generation
- PM2 For process management
- Express
- React

## Start

To start the server and react app run:

```
./start.sh
```

Connect to the server at http://localhost:3000

# Stop

To shutdown the process run:

```
./stop.sh
```
